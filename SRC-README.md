# SRC - Código de Visualización Cuatro Flor

## Propósito
Código fuente para visualizaciones interactivas de ciclos planetarios y calendarios mesoamericanos.

## Estructura
```
src/
├── visualization.html      # Visualización principal de ciclos
├── planet_harmonics.py     # Cálculo de armónicos planetarios
├── csv_to_viz.py          # Conversión CSV a visualización
├── fetch_and_render.py    # Fetch y render de datos
├── index.html             # Dashboard principal
├── output_harmonics.csv   # Datos de salida de armónicos
├── planetary_harmonics.html # Visualización de armónicos planetarios
├── planetary_harmonics_dynamic.html # Versión dinámica
└── visualizer.js          # Librería de visualización JS
```

## Uso
Los archivos HTML pueden abrirse directamente en navegador.
Los scripts Python requieren dependencias estándar (pandas, matplotlib, etc.)

## Nota
Esta carpeta contiene código de visualización opcional.
La investigación principal está en `/research/calendarios-mesoamericanos/`
